St-Hunter.
Оригинал на оффициальном ГитХабе Фила.


Для работы обязательно нужен ELFPack 3.0!!! 
Перед обновление читать: История коммитов!!! 
https://github.com/FTCr/Siemens/commits/master

Основное: 
Shell - оболочка 
В конфиге нужно указать путь к скину. По умолчанию 0:\Zbin\Shell\Skins\default\ 
Очередь загрузки плагинов соответствует их именам. 
libs - стандартная библиотека, в которой собраны основные функции. 
libshell - библиотека оболочки 
Плагины: 
DClock - цифровые часы 
IIH - отображение оператора и даты в заголовке 
MainTab - софт клавиши на главной вкладке 
TFGui - "Выключение" 
TFGui2 - альтернативная реализация "Выключение" 
SInbox - пропущенные события по софткею. 
EInfo - дополнительная информация 
---------------------- 
MesTab - вкладка "Сообщения" 
---------------------- 
BrowserTab - вкладка "Интернет" 
---------------------- 
MPTab - вкладка "Медиаплеер" 
---------------------- 
SetTab - вкладка "Настройки" 
---------------------- 
DeskSwitcher - переключение рабочих столов 
Для добавления рабочих столов нужно добавить парные иконки. 
KeyLocker - блокировщик клавиатуры. 
Обязательно для работы должна быть подгружена swi.blib!!! 
DGui - набор номера 
BKeys - горячие клавиши 
PrGui - плагин для выбора профиля 
Скины: 
Стандартный скин 
Лэнгпак забираем тут. 
Исходники на GitHub.

https://github.com/FTCr/Siemens.git

SKIN_DIR - директория скина 
SHELL_DIR - директория откуда запущен эльф 
Структура директорий: 
SHELL_DIR\Langpack\ - директория ленгпаков 
SHELL_DIR\Plugins\ - директория плагинов 
SHELL_DIR\Skins\SKIN_DIR\ - директория скина 
Структура скина: 
SKIN_DIR\Configs\ - директория настроек плагинов 
SKIN_DIR\s-col.bcfg - файл настроек цвета 
SKIN_DIR\s-coord.bcfg - файл настроек координат 
SKIN_DIR\s-font.bcfg - файл настроек шрифтов 
SKIN_DIR\Img\ - директория графики 
SKIN_DIR\Img\bottom.png - фон софтклавиш 
SKIN_DIR\Img\cursor.png - курсор, используется в менюшках 
SKIN_DIR\Img\header.png - фон заголовка
