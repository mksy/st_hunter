JKeyEmul - эмулятор софт-клавиш и джойстика для Java

Описание:

Эльф для эмуляции софт-клавиш и джойстика/крестовины от другого телефона в Java-приложениях, когда это необходимо. Захват клавиш (кейхук), а значит и эмуляция работает только когда Java-мидлет "наверху". Сделано для избежания разных курьёзных ситуаций. Необходимо ввести Адрес JAVA CSM для ваших телефона и прошивки. В стандартных формах, текстовых полях и т.д. следует вручную отключать эмуляцию. Адрес "A0000000" для теста, т.е при нём эмуляция будет, но не только в Java, но и везде.
