 Версия 2.0 - Теперь с редактором.

 Шрифт 4*8, 6*8, 8*8. Для редактирования шрифтов используется программа Siemens Character Editor - установить
размер шрифта 8(width) на 7(height) и снять все галочки. Для шрифта 4*8 символы должны дублироваться по 2 в одном
знакоместе - увидите сами при загрузке уже сделаного 4.fnt.

 Для просмотра файла откройте его CardExplorer'ом - тип файла должен быть неизвестен эксплореру.

 !!! Не стоит открывать ted.tmp - может случиться жопа. Тоже относится и к файлу A:\TED\history.

 !!! При закрытии записывает файл A:\Trash\TED.snapshoot, который используется для быстрого восстановления последнего файла
(например, если TED был закрыт входящим звонком). Для этого достаточно просто запустить TED не через ассоциации файлов
(открыв TED.bin эксплорером), а после входящего звонка, если вы пользуетесь BFL или UNI2 - просто нажмите зеленую трубку в основном меню.
ВНИМАНИЕ: для правильной работы временные файлы и история не должны быть нарушены! После жабы зеленую трубку лучше не нажимать,
откройте бинарник эксплорером!

 Управление: 

 Стрелки - перемещение курсора. Если курсор выключен - перемещается текст по вертикали по 1 строке,
по горизонтали - по 8 символов. Долгое нажатие влево/вправо - переход по словам

 Громкость - на страницу вверх/вниз.

 Правый софт при просмотре - меню из пунктов:
 Font size=4 - установить размер шрифта 4*8
 Font size=6 - установить размер шрифта 6*8
 Font size=8 - установить размер шрифта 8*8
 Direct load - загрузить файл, как он есть (без форматирования). Строки длиннее 255 символов разрезаются.
 Load & Format - загрузить файл с форматированием. Абзацем считается набор строк (признак окончания -
0x0A, 0x0D, 0x0D0A, 0x0A0D) в котором первая строка начинается
с пробела, остальные строки не пустые и начинаются с любого другого символа.
 EOL->EOL SPC - загрузить файл с форматированием. Абзацем считается каждая строка текста.
 Extend Lines - включить/выключить расширение строк пробелами до размера экрана.
 Save as... - записать текущий файл.

 Тип форматирования и размер шрифта запоминается в истории для
каждого файла. Если файла в истории нет - сразу при загрузке открывается это меню.

 Правый софт при редактировании - меню из пунктов:
 Insert line - вставить пустую строку
 Delete line - удалить строку
 Split line - разрезать строку в позиции курсора
 Join lines - склеить текущую и следующую строку
 Insert time - вставить время в виде ЧЧ:ММ
 Insert date - вставить дату в виде ДД:ММ:ГГГГ
 Paste - вставить запомненые строки из клипбоарда (пока не работает)
 Clear clipb - очистить клипбоард (пока не работает)

 Левый софт при просмотре - меню из пунктов:
 Goto...
 -Top - переход на начало текста
 -Line... - переход на строку по номеру
 -Percent... - переход на строку по проценту текста
 -Bottom - переход на конец текста
 -Last Saved - переход на позицию из истории файла
 -BM1 - переход на закладку 1
 -BM2 - переход на закладку 2
 -BM3 - переход на закладку 3
 -BM4 - переход на закладку 4
 Set Bookmark...
 -BM1 - установить закладку 1
 -BM2 - установить закладку 2
 -BM3 - установить закладку 3
 -BM4 - установить закладку 4
 Search...

 Левый софт при редактировании - backspace

 1 - пусто
 2 - при просмотре - на страницу вверх
 3 - пусто
 4 - при просмотре - на слово влево, включает курсор
 5 - при просмотре - включение/выключение курсора
 6 - при просмотре - на слово вправо, включает курсор
 7 - при просмотре - в начало строки, включает курсор
 8 - при просмотре - на страницу вниз
 9 - при просмотре - в конец строки, включает курсор
 0 - информация
 * - пусто
 # - Переключение кодировок DOS/WIN/КОИ8

 Красная трубка - выход.

 Зеленая трубка - переключение в режим редактирования, автоматически включается курсор. Выход из режима редактирования -
красная трубка. Редактор - как обычный ввод текста, при этом работают стрелки вверх, вниз, громкость как и в режиме
просмотра.
